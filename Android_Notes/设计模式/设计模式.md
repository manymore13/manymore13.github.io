
# 设计模式分类

- **创建型模式**（如工厂模式、单例模式、建造者模式）  
  - 关注对象的创建过程，提供灵活的对象实例化方案。

- **结构型模式**（如适配器模式、装饰器模式、代理模式）  
  - 处理对象之间的组合关系，提供更灵活的结构。

- **行为型模式**（如观察者模式、策略模式、命令模式）  
  - 关注对象间的交互与职责分配，使系统具有更好的灵活性和扩展性。
